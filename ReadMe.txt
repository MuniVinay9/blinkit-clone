Blinkit Clone - Responsive Web Design using Bootstrap

Project Overview

This project is a Blinkit Clone, built using HTML, CSS, and Bootstrap, with a focus on responsive web design. The goal is to replicate the look and feel of Blinkit's online grocery delivery platform while ensuring seamless user experience across different devices.

Features

Fully responsive design using Bootstrap 5

Clean and modern UI similar to Blinkit

Mobile-first approach for enhanced usability

Bootstrap Grid System for a well-structured layout

Custom CSS Styling for refinements beyond Bootstrap

Sticky Navbar for easy navigation

Hero Section with search bar functionality (UI only)

Footer Section with links and social icons
